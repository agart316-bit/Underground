Foundation

Glass Levels

Three elevation levels create visual hierarchy. Higher levels are more opaque, more blurred, and carry stronger shadows — they feel closer to the user. Every surface in the app must declare one of these three levels.

L0 · Ambient
Page atmosphere, decorative rects, hover zones, inactive filter regions
Fillrgba(255,255,255,0.06)
Blur8px sat 140%
Borderrgba(255,255,255,0.14)
Shadownone
L1 · Surface
Cards, panels, nav bar, side panels, filter chips, form inputs, buttons
Fillrgba(255,255,255,0.18)
Blur20px sat 180%
Borderrgba(255,255,255,0.28)
Shadow0 8px 32px + inner sheen
L2 · Elevated
Modals, drawers, focused overlays, tooltips, profile previews
Fillrgba(255,255,255,0.72)
Blur32px sat 200%
Borderrgba(255,255,255,0.55)
Shadow0 20px 56px rgba(48,24,72,.14)
Rule: Never nest the same glass level inside itself. An L1 card inside an L1 panel reads as flat. Go L0 → L1 → L2 as you layer content inward. Modals (L2) on top of cards (L1) on top of page (L0).

Interaction

Button Hierarchy

Four tiers, strictly enforced. Only one Tier 1 button should appear in any given view. Tier 2 and 3 can coexist. Tier 4 (ghost) is for dismissal, back navigation, and low-friction links.

Tier 1 Primary
The one unmissable action on the screen. Deep fuchsia fill, strong glow shadow, sheen overlay. One per view max.
Post · Submit · Publish · Send
Tier 2 Social
Social-layer actions. Soft pink glass — clearly interactive but subordinate to Tier 1. Can appear next to Tier 1.
Follow · RSVP · Save · Connect · Like
Tier 3 Glass
Secondary actions that need to exist but shouldn't pull focus. Neutral glass, white border, no color.
Details · View · Edit Profile · Filter · Share
Tier 4 Ghost
Inline navigation, dismiss, back, and low-stakes text actions. Underline signals it's tappable without competing for attention.
← Back · Cancel · Skip · Learn more
Icon Button
Circular glass button for heart/save, close, share. Use with a visible tooltip or clear context — no orphaned icons.
♡ · ✕ · ↗ · ···
Decision rule Ask: "What's the single most important action here?" That one gets Tier 1. Everything else steps down. If you're unsure, default to Tier 3.

Navigation

Nav & Filters

Navigation uses a pill container (L1 glass) that groups items visually. The active item gets a distinct inner pill (L2 glass) — a clear selected state without needing bold text or color.

Global Nav — site header

Active: inner pill rgba(255,255,255,0.42) + weight 600 · Hover: rgba(255,255,255,0.22) · Default: transparent

Filter Chips — content tabs / page filters

Active: rgba(255,255,255,0.52) + weight 600 · Default: rgba(255,255,255,0.14) · These should sit on the page background, not inside a container

Key fix: Previously nav links, filter chips, and buttons all used the same look. Now: nav sits in a pill container (structural), filters are loose chips (contextual), buttons use the tier system (action). Three clearly different roles, three clearly different shapes.
